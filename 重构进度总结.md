```

├── components/
│   ├── ui/              # 基础 UI 组件
│   ├── layout/          # 布局组件
```

在 `astro.config.mjs` 中配置：
- `@/` → `./src`
- `@components/` → `./src/components`
- `@lib/` → `./src/lib`
- `@types/` → `./src/types`
- `@styles/` → `./src/styles`

## 🎨 Tailwind 设计系统

### 颜色规范
| 用途 | 颜色值 | Tailwind 类示例 |
|------|--------|----------------|
| 主色调 | `#e74c3c` | `text-[#e74c3c]`, `bg-[#e74c3c]` |
| 背景色 | `#161618` | `bg-[#161618]` |
| 卡片背景 | `#242426` | `bg-[#242426]` |
| 表面色 | `#1e1e20` | `bg-[#1e1e20]` |
| 主文字 | `#e8e8e6` | `text-[#e8e8e6]` |
| 次要文字 | `#a8a8a6` | `text-[#a8a8a6]` |
| 弱化文字 | `#565658` | `text-[#565658]` |
| 边框 | `#303032` | `border-[#303032]` |

### 响应式断点
使用 Tailwind 默认断点：
- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px
